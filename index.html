Как “развернуть” на GitHub Pages

  1. Зайди в репозиторий → Settings
  2. Открой Pages
  3. В Build and deployment выбери: Deploy from a branch
  4. Branch: main, Folder: /(root) → Save
  5. GitHub покажет URL сайта (обычно через 1–2 минуты)
